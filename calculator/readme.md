 # Simple Calculator using Python

A simple command-line calculator built using Python. It performs basic arithmetic operations using the `match-case` statement and includes exception handling for invalid inputs.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Uses `match-case`
- Exception handling
- Prevents division by zero

## Concepts Used

- Variables
- User Input
- Match-Case
- Exception Handling
- Arithmetic Operators

## How to Run

1. Clone the repository:
   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project folder:
   ```bash
   cd Python-Simple-Calculator
   ```

3. Run the program:
   ```bash
   python calculator.py
   ```
